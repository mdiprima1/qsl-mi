# QS Lab MI Research Package — Commands

**Package version: 1.0.0**

These are the commands installed into your Claude Code by the QS Lab MI Research Package. Type them in any Claude Code session.

| Command | What it does | Added in |
|---|---|---|

More commands are added as the course grows. To get new commands, re-run the install prompt (see `INSTALL.md`) — it replaces your installed commands with the current version.
